# Granny's Helper: Anti-Scam Protection Script

**Protect vulnerable users from tech support scams and malware with this Windows batch script.**

## What It Does
- **Browser Hardening**: Disables developer tools and forces installation of uBlock Origin (a powerful ad/malware blocker) in Chrome, Edge, and Firefox.
- **Shell Restrictions**: Blocks execution of Command Prompt, PowerShell, and Windows Terminal to prevent scammers from running malicious commands.
- **Automatic Enforcement**: Restarts browsers after configuration and reboots the computer with a 60-second countdown to apply all changes.

## Keywords
anti-scam, elderly protection, browser security, malware prevention, tech support scam blocker, uBlock Origin, Windows security script, registry hardening, command line restriction, scam protection for seniors.

## Usage
1. Run `GrannysHelper.cmd` as administrator.
2. The script will detect installed browsers, apply policies, and reboot the system.
3. uBlock Origin will block scam sites, pop-ups, and trackers.

## Requirements
- Windows 10/11
- Admin privileges
- Internet connection for extension downloads

## Disclaimer
This script modifies system policies and reboots the computer. Test in a safe environment. Not foolproof against advanced threats—combine with user education and antivirus.
